# Emojis Awesome

気軽に絵文字を使えるCSSライブラリーです。

## Setup

```
# npm
npm i -D emojis-awesome
# yarn
yarn add emojis-awesome
```

```
<link rel="stylesheet" href="css/emojis-awesome.css">
```
